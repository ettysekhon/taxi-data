
## Enable the AI Agent

Add the ai: snippet to the KESTRA_CONFIGURATION part in the docker-compose.yml file

```
environment:
      KESTRA_CONFIGURATION: |
        datasources:
          postgres:
            url: jdbc:postgresql://postgres:5432/kestra
            driverClassName: org.postgresql.Driver
            username: kestra
            password: k3str4
        kestra:
          ai:
            type: gemini
            gemini:
              api-key: YOUR-API-KEY-HERE
              model-name: gemini-2.5-flash
```
